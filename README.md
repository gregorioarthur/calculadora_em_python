# Calculadora básica em Python

Este projeto consiste em uma calculadora que é capaz de realizar as seguintes operações:

- Soma e subtração
- Multiplicação e divisão
- Exponenciação e raíz

## O que foi praticado
- Uso de funções
- Tratamento de erros
- Estruturas condicionais

## Tecnologias
- Linguagem: Python
- Ferramentas: Nenhuma

## Estrutura
Todo o código está contido em "main.py", onde ocorre a execução do programa.

## Como executar
'''bash 
python main.py
